## 📌 About Project

> 💡 A simple yet powerful **Phone Book Management System** built using **C++ OOP + File Handling**

✨ Features:
- 📞 Add new contacts  
- 🔍 Search existing contacts  
- 💾 Data stored permanently in file  
- ⚡ Fast and simple console interface
